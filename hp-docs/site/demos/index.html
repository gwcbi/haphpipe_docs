Demo01

We provide a subsample of reads for this sample. We obtained this sample using fastq-dump.

fastq-dump --origfmt --defline-qual "+" --split-files --accession SRR8525886

condo activate haphpipe
hp_sample_reads --fq1 SRR8525886_1.fastq --fq2 SRR8525886_2.fastq --nreads 10000

Demo01 uses real HIV-1 data (SRA: SAMN10820316). This demo runs both the haphpipe_assemble_01 de novo assembly and haphpipe_assemble_02 amplicon assembly pipelines (see here for details).
